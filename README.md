# 🎮 Tambola Game - Social Play

A web-based multiplayer Tambola (Housie) game built with Streamlit for playing with friends and family.

## 🎯 Features

- ✅ Create/Join game rooms with unique codes
- ✅ Play with computer when no friends available
- ✅ Random ticket generation for each player
- ✅ Manual number marking with undo option
- ✅ Random number calling (1-90)
- ✅ Multiple winning patterns (Jaldi 5, Lines, Four Corners, Full House)
- ✅ Multiple Full House winners (1-4)
- ✅ Real-time progress tracking
- ✅ Chat system with emojis
- ✅ Winners board
- ✅ 100% legal - no real money involved
